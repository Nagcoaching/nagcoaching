# 11 — Perf, accessibilité, technique

## Performance
- LCP < 2.5s (images hero optimisées)
- Lazy-load galerie
- Fonts : 2 familles max, subset, preload
- Compression images (WebP/AVIF)

## Accessibilité
- Contraste AA
- Navigation clavier
- Labels formulaire
- Focus visible
- Alt text images

## Tech (au choix)
### Option A — Webflow
+ rapide à livrer, facile à éditer
- coût mensuel

### Option B — WordPress (block theme)
+ flexible, bon SEO
- maintenance plugins

### Option C — Next.js
+ perf top, contrôle total
- besoin dev

## Sécurité & légal
- Mentions légales + politique confidentialité
- RGPD : cookies (si tracking), bandeau + consentement
