# 03 — Direction Artistique (DA) — moderne, premium, local

## Intentions
- **Premium accessible** : “studio privé” = service haut de gamme, mais ambiance simple.
- **Confiance** : clair, lisible, peu de blabla, beaucoup de preuves.
- **Énergie** : sport = mouvement, mais sans agressivité “bodybuilder”.

## Mood (références)
- Photos en lumière naturelle + détails matériel + scènes réelles (pas stock clichés).
- Fond clair ou très sombre, mais lisibilité maximale.
- Grands titres, beaucoup d’air, sections courtes.

## Palette (proposée)
> Ajustable selon logo existant.
- Base : Noir/Graphite (texte, contrastes)
- Fond : Blanc cassé / Gris très clair
- Accent : une couleur unique (ex: bleu électrique OU vert “énergie”) pour CTA
- Neutres : gris 100–900 (scale)

## Typographies (proposées)
- Titres : sans-serif moderne (ex: **Sora**, **Space Grotesk**, **Poppins**)
- Texte : sans-serif très lisible (ex: **Inter**, **DM Sans**)
- Règle : titres impactants, paragraphes courts.

## Iconographie
- Icônes ligne (outline) simples, cohérentes (Lucide / Phosphor).
- 1 style unique, pas de mélange.

## Photos / vidéo
- Priorité : photos réelles du studio + coach (authenticité).
- 1 vidéo courte “visite du studio” (20–40s) en hero ou section.
- Formats nécessaires :
  - Hero : 1920×1080 (ou 2400×1350)
  - Cartes : 1200×900
  - Galerie : 1200×1200
- Traitement : contraste doux, pas de filtres agressifs.

## Layout & grid
- Desktop : container 1200–1280 px, grille 12 colonnes.
- Mobile : 1 colonne, CTA toujours visible (sticky possible).
- Sections : alternance “texte + média” pour rythme.

## Micro-interactions
- Hover subtil (élévation, légère translation, underline).
- Animations à l’entrée : 150–250 ms, discret, pas de “sur-animation”.
- Boutons : états normal/hover/active/disabled.

## Accessibilité
- Contraste AA minimum.
- Taille texte base 16–18px.
- Focus states visibles clavier.
