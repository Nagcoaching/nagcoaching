# 08 — SEO local (et pages géolocalisées)

## Objectif
Apparaître sur les requêtes :
- coach sportif + ville
- studio coaching / personal trainer + zone
- coaching privé + Melun / Carré Sénart / Vert-Saint-Denis

## Quick wins
1) Optimiser titres/H1 des pages principales avec la zone
2) Ajouter une page “Avis” + données structurées
3) Créer 4–8 landing pages locales (qualité > quantité)
4) Ajouter schémas (LocalBusiness / SportsActivityLocation / FAQ)

## Pages locales (exemples)
### `/coach-sportif-vert-saint-denis`
- Intro : studio privé + proximité
- Preuves : avis + photos + plan
- FAQ locale (parking, accès)

### `/coach-sportif-melun`
- Angle : “à X minutes de Melun”
- Itinéraire + parking + horaires

### `/coach-sportif-carre-senart`
- Angle : “studio privé proche Carré Sénart”
- Itinéraire, créneaux (matin/soir)

## Contenu recommandé par page locale
- 700–1200 mots max, facile à lire
- 6 photos réelles min (studio + coach)
- 3 avis pertinents (si possible avec ville)
- 1 bloc “comment réserver”
- 1 FAQ (6 questions)
- Map embed + NAP (Name Address Phone) identique partout

## Schema (données structurées)
- `LocalBusiness` ou `HealthAndBeautyBusiness` (selon choix technique)
- `FAQPage` sur pages avec FAQ
- `BreadcrumbList`
- `WebSite` + `Organization`

## Base technique SEO
- URLs propres
- Title tags uniques
- Meta descriptions orientées action
- Images compressées + alt text
- Sitemap.xml + robots.txt
- Redirections 301 depuis anciennes pages

## Blog (option)
Si blog relancé : 1 article / mois minimum, sinon éviter.
Thèmes utiles localement :
- “Comment choisir un coach sportif à Melun”
- “Studio privé vs salle : avantages”
- “Reprise après blessure : erreurs à éviter”
