# 05 — Composants (UI) — specs fonctionnelles

## 1) Header
- Logo gauche, nav centre/droite, CTA “Réserver” accent
- Sticky au scroll (option)
- Mobile : burger + CTA visible

## 2) Hero
- H1 local + promesse
- Sous-titre en 2 lignes max
- 2 CTA : “Réserver un essai” (primary) / “Voir le studio” (secondary)
- Média : photo/vidéo courte

## 3) Offer cards (Solo / Duo / Small group / Distance)
- 4 cartes max
- “à partir de …” + durée + pour qui
- CTA vers page détail

## 4) Proof strip
- 3 à 5 blocs : avis, proximité, studio privé, flexibilité, suivi

## 5) Section “Studio”
- Galerie (6–12 images)
- Bullets : 70 m², 1–4 pers, équipements clés, accès/parking

## 6) Testimonials
- 6 avis visibles + lien “Voir tous les avis”
- Important : inclure prénom + objectif + ville si possible

## 7) Pricing table
- Très lisible : lignes (Essai / Séance / Pack)
- Badge “le plus choisi”
- CTA sous la table

## 8) FAQ accordion
- 8–12 questions
- SEO-friendly (FAQ schema)

## 9) Contact module
- Formulaire court (Nom, Téléphone, Objectif, Disponibilité)
- Click-to-call + WhatsApp
- Map + adresse + horaires

## 10) Footer
- Coordonnées + réseaux + pages légales
