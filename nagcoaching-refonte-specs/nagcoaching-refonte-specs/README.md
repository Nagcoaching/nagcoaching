# Refonte nagcoaching.fr — dossier de specs (designer + intégrateur)

Ce dossier contient toutes les spécifications (UX, DA, contenu, SEO local, composants, tracking) pour produire une refonte moderne du site **Nag Coaching**.

## Objectif business
- Générer plus de demandes **locales** (rayon ~15 min en voiture autour du studio).
- Augmenter la conversion : visite → prise de contact / réservation.
- Créer des landing pages efficaces pour **Google Ads** + SEO local.

## Livrables attendus
- Design system (tokens + composants) + maquettes desktop & mobile.
- Intégration (Webflow / WordPress / Next.js, au choix).
- Mise en place SEO local + tracking (GA4/Ads/Meta + events).

## Structure du dossier
- `01_strategy.md` : cibles, proposition de valeur, messages.
- `02_sitemap_ia.md` : arborescence + IA (navigation).
- `03_design_direction.md` : DA moderne, mood, typos, couleurs, iconographie.
- `04_design_system_tokens.md` : variables + règles (spacing, radius, etc.).
- `05_components.md` : composants UI (specs + états).
- `06_page_home.md` : wireframe + contenus (copy).
- `07_page_service_templates.md` : gabarits pages service (solo/duo/small group/distanciel/entreprise).
- `08_local_seo.md` : plan SEO local + pages géolocalisées + schema.
- `09_content_playbook.md` : tone of voice + bibliothèque de preuves + FAQ.
- `10_tracking_analytics.md` : plan de tracking & conversions.
- `11_perf_accessibility.md` : perf, accessibilité, technique.
